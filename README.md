# 🚀 JavaScript - Lógica de Programação

Projeto desenvolvido durante os cursos de **JavaScript** da Alura com o objetivo de praticar os principais conceitos da linguagem por meio de desafios e de um projeto prático: o **Jogo do Número Secreto**.

## 📚 Conteúdos praticados

- Variáveis (`let` e `const`)
- Operadores
- Condicionais (`if`/`else`)
- Estruturas de repetição (`while`)
- Funções
- Arrays
- Manipulação do DOM
- Eventos
- Geração de números aleatórios
- Boas práticas de organização do código

## 🎮 Projeto Principal

### Jogo do Número Secreto

O usuário deve adivinhar um número gerado aleatoriamente pelo sistema.

### Funcionalidades

- ✅ Geração automática de número secreto
- ✅ Controle de tentativas
- ✅ Dicas informando se o número é maior ou menor
- ✅ Reinício da partida
- ✅ Evita repetição de números sorteados
- ✅ Utilização da biblioteca ResponsiveVoice para síntese de voz
